# Vidly

A video rental application.


## Project Setup

1. Clone the project
```
git clone https://github.com/MohAhm/myreads.git
```

2. Change into the app directory
```
cd myreads
```

3. Install the dependencies
```
npm install
```

4. Start the application
```
npm start
```
